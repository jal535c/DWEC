# DWEC
PongMulti: Videogame multiplayer using node.js, express, socket.io and P5.js library

Instructions:
- clone the repository
- cd pongmulti
- npm install
- node server.js
- Go to the browser at http://localhost:3000

Deploy on Heroku at:
https://pongmultip5.herokuapp.cpm
